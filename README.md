🟣 IT Ticketing & SLA Management Simulation (Jira)
📌 Description

Simulated real-world IT service management (ITSM) workflows using the Jira platform to create, categorize, and resolve technical support tickets in accordance with defined Service Level Agreements (SLAs). The project demonstrates incident handling, prioritization, escalation processes, and dashboard reporting aligned with enterprise IT operations.

🎯 Key Features

Ticket lifecycle management

SLA-based incident prioritization

Issue categorization and escalation

JQL-based dashboard creation

🧠 Skills Demonstrated

ITSM Workflow Simulation

Incident Response Processes

SLA Monitoring

SOC Support Operations

Operational Security Documentation
