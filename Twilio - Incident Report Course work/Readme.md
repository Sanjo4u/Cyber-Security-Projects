🟣 Twilio Social Engineering Incident Analysis
📌 Description

Performed post-incident analysis of the 2022 breach involving Twilio by mapping attacker tactics across the Cyber Kill Chain and MITRE ATT&CK frameworks. The project evaluated exploited vulnerabilities and human factors contributing to initial compromise.
