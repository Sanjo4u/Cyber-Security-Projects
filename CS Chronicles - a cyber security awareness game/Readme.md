MSc Dissertation – Cybersecurity Awareness Game
📌 Description

Designed and developed an interactive cybersecurity awareness training tool using the Twine platform, incorporating real-world attack vectors and endpoint security controls. The system analysed user behaviour through Python-based data collection and demonstrated measurable improvements in organisational security awareness metrics.

🎯 Key Features

Gamified phishing & endpoint scenarios

Behaviour tracking using Python

User engagement analytics

Interactive training delivery

🧠 Skills Demonstrated

Human-Centric Security

Security Awareness Training

Endpoint Threat Modelling

Behavioural Data Analysis

Risk Reduction Strategies
